# text-search-engine-m1
Implémentation complète d’un moteur de recherche basé sur un index inversé et le scoring TF‑IDF. Construction de l’index, parsing, normalisation, ranking, benchmarks, dataset synthétique, et interface Streamlit. Projet réalisé pour le mini‑mémoire M1

Dataset du projet:
https://www.kaggle.com/datasets/karnikakapoor/ml-in-healthcare-patent-data
